Je suis un paragraphe avec un span qui a comme display par défaut : inline

Les liens aussi sont en inline par défaut

Toutes les balises stylistiques qu'on a déjà vu dans une vidéo, la balise u, la balise i, etc

Par contre, la

div est en block par défaut

Il ne faut jamais mettre de balise de titre dans un paragraphe mais pour l'exemple, on peut voir que cette dernière est en

block par défaut

La balise P est aussi en

block par défaut

Aucun élément HTML n'est en inline-block par défaut

Cependant, on peut mettre ce lien en inline-block et comparer son comportement avec ce dernier en inline
